What it evaluates

Memory as an observable lifecycle, not a final accuracy score.

WorldMemArena turns long-horizon multimodal interaction into stage-level diagnosis, so failures can be localized instead of hidden inside one QA number.

1

Observe to Write

Can the agent identify future-useful evidence from session observations, actions, images, and feedback?

2

Update and Consolidate

Can memory revise stale facts, remove outdated entries, and preserve a consistent world state over time?

3

Retrieve and Use

Can the system surface the right evidence and faithfully use it for later answers and decisions?

Data construction pipeline across Lifelong Evolution and Agentic Execution.
Two regimes

Controlled state evolution plus realistic agent trajectories.

Lifelong Evolution tracks personal and task states across sessions. Agentic Execution reconstructs memory from real observations, actions, screenshots, and environment feedback. Together they stress both persistent state tracking and action-grounded experience reuse.

Main takeaways

Better storage is not the same as better memory.

Experiments compare long-context agents, manually designed memory systems, RAG systems, and harness-based memory agents.

1

Memory coverage and QA quality are misaligned.

Systems can store many correct memories but fail to retrieve or use the specific evidence needed at answer time.

2

Visual memory remains a bottleneck.

Complex multimodal tasks degrade when visual evidence is compressed into incomplete text proxies.

3

Agentic trajectories expose brittleness.

Useful information is distributed across actions, feedback, GUI states, and screenshots, not just dialogue.

4

Harness memory is flexible but costly.

Agent-managed memory adapts better in some regimes, yet remains less stable and more expensive.
